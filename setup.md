# Windows:

## from root:
```
py -m venv Chatbot
Chatbot/Scripts/activate
pip3 install tensorflow
pip3 install numpy

py training.py

```

# MacOS:

## from root:
```
python3 -m venv Chatbot
source Chatbot/bin/activate
pip3 install tensorflow
pip3 install numpy
pip3 install nltk
python3 training.py
python3 chatbot.py
```
